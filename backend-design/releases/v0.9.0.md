# FocusLink v0.9.0

> 本地候选构建日期：2026-07-13
>
> 状态：未发布；仓库没有 `v0.9.0` tag 或可验证的精确对应提交，以下仅记录本机候选资产。

## 本次更新

### 四区主工作面与明亮主题

- 主导航统一为专注、任务、统计、设置，删除重复标题、装饰性空卡和分散入口。
- 新的高对比明亮主题成为默认，保留深色主题，并提供六套强调色。
- 页面切换、状态光场、按钮按压、列表入场和任务完成反馈重新打磨；系统要求减少动效时自动退化。
- 修复快速切换页面时旧页面可能停在退出态、导航状态与可见内容不一致的问题。

### 独立任务工作台

- 集中浏览本地、dida CLI 与 TickTick OAuth 任务，支持项目、搜索、未完成/已完成/全部和层级任务。
- 支持本地快速创建、完成、取消完成、关联任务与关联后立即开始专注。
- 普通滴答任务、checklist 子项和本地任务使用同一套可逆状态 API；checklist 只更新父任务 `items` 中的目标项。
- 当前 dida CLI 缺少普通任务恢复参数时，使用既有 CLI 凭据的最小 Open API 兼容桥，并在写入后回读确认；凭据不会进入日志、IPC 或界面。
- 兼容 Dida 恢复任务后仍保留历史 `completedTime` 的行为，以显式 `status=0` 为未完成权威状态。

### 双同步可靠性

- dida 继续使用 comment-first、正文回退和稳定 segment marker；中文、换行与 JSON 全部通过 argv 传递。
- 番茄 To-do 继续本地先写、云端确认后再标记已同步；未知内容统一进入“学习”。
- 任务关联、滴答同步与番茄 To-do 云同步分别展示，不再用一个模糊状态互相代替。
- 发布候选已用临时真实数据验证 dida 任务完成/恢复与番茄 To-do 本地/云端幂等写入，测试数据均已清理。

### 边缘进度小窗

- 展开态固定为 `256×92`，在一块高密度控制台中容纳任务、当前片段、累计专注/暂停/总历时与核心控制。
- 收起态固定为 `184×35`，只保留当前状态、时间、对应累计和展开入口。
- 支持左、右、上、下四边吸附：14px 进入、30px 离开，贴边 260ms 自动收起，拖离 140ms 自动展开，点击展开提供 900ms 防回弹。
- 深浅主题、运行/暂停、多显示器 work area、透明边界和 Windows 缩放均纳入真实 Electron smoke。

### 项目与发布规范

- 前端与后端源码分别收敛到唯一实现树；设计文档只保留 `frontend-design/` 与 `backend-design/` 两个入口。
- 新增面向后续 AI 的单一真相规范与接手清单，删除平行文档树、旧 UI 源码和一次性修复报告。
- 新增 tag 驱动的 GitHub Actions 发布闭环：版本校验、检查、打包、SHA256、GitHub Release 与三项附件。

## 升级提示

- 数据库迁移保持幂等，原有专注、暂停、关联和同步队列会继续保留。
- 已设置深色主题的用户继续使用深色；全新安装默认使用明亮主题。
- dida 普通任务“取消完成”的兼容桥要求本机 dida CLI 已登录并存在有效配置；仅在 CLI 本身没有该能力时启用。
- Windows 150% 缩放下，`184×35` 的可见胶囊可能取整为 36px；透明原生外框受系统最小高度影响，但不会放大可见内容。

## 已知限制

- 番茄 To-do 真云确认要求其桌面客户端已登录；客户端关闭时记录会安全留在本地待上传队列。
- 2026-07-13 回读确认：仓库的 GitHub Releases 页面仍为空，历史 tag 到 `v0.5.0` 为止；v0.6.0–v0.9.0 只有本地候选资产且没有准确版本提交。本版本不补造 tag 或 GitHub Release，完整 tag/Release/附件/回读闭环从后续可验证版本开始执行。
