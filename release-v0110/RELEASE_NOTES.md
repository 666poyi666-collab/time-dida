## 本次更新

### 动态材质工作面

- 主窗口加入低对比环境光场、轨道轮廓、稀疏粒子、状态呼吸光和强调色伴生色，专注、任务、统计与设置统一为高不透明材质面、折射边缘和定向阴影。
- 六种强调色会同时改变导航、主要按钮、环境层与状态细节；所有持续动画在“减少动态效果”下停用。
- 全屏页面仅做稳定淡入，不动画整页滤镜或 blur；截图回归中发现的 Electron 黑色合成块已消除。

### 双字体体验

- 默认新增“澄澈”字体：Manrope Variable 搭配 Noto Sans SC Variable。
- v0.10 的 Geist“精准”字体完整保留，可在设置 > 体验即时切换；选择会持久化，不触发无关设置副作用。

### 过渡式边缘小窗

- 小窗仍严格只有 184×35 与 256×92 两种尺寸。
- 折叠时间增至 25px、展开时间增至 31px；折叠态底部新增 3px 真实专注占比进度轨。
- 拖拽释放后先吸附，再播放 320ms 收束/淡缩反馈后折叠；过渡期间再次拖动会取消折叠，不抢鼠标。

### 单一源码工作区

- renderer、Electron、shared、测试、脚本、设计文档和构建配置全部收入 `FocusLink/`。
- 仓库根目录只保留 GitHub/治理入口、`FocusLink/` 与最近三个 `release-v*`；GitHub Actions、Electron Builder、smoke 路径和文档链接已同步迁移。

## 修复

- 修复全屏 blur、route transform 与透明顶栏组合可能造成的 Electron 黑色合成块；任务/统计大壳不再执行整块 transform，底层材质改为不透明。
- 字体切换归入主题变更域，不会误触快捷键重注册、开机启动或同步设置。
- 小窗使用 Windows 原生 move loop 识别真正释放，按住不动不改尺寸；贴边折叠从突变改为可见且可取消的过渡，并容忍多显示器、负坐标与 DPI 的 1–2px 归一化。
- 浅色主题主要操作渐变改用通过文字对比度门槛的深色端点；环境粒子不再永久占用合成层。
- 滴答任务完成/恢复、6 秒撤销、强制刷新、评论优先 marker 幂等同步、番茄 To-do 学习分类、统计 request-id 保护及三时间账本语义保持不变。
- 番茄 To-do 手动同步在客户端未运行时可用参数数组和 `--remote-debugging-port=0` 按需启动，目标通过标题与特征 API 身份校验后才连接；已普通运行但无桥时绝不杀进程或重启，界面会要求完全退出后再连接，后台周期也不会擅自启动外部应用。
- 番茄 To-do 上传 success 只表示“上传已确认”；当前客户端仍无专注记录的独立云端回读或远端删除能力。

## 升级提示

- 用户数据库、任务关联、计时记录和同步队列无需迁移。
- 旧设置缺少 `fontProfile` 时自动使用新的“澄澈”方案，可随时切回 Geist“精准”。
- 开发与 Web Coding 的工作目录改为仓库内 `FocusLink/`；正式发布资产仍位于仓库根级 `release-v0110/`。

## 已知限制

- 安装器最终启动 smoke 需在不打断现有暂停专注的前提下执行；GitHub tag、Release 与线上资源回读尚未进行。

- 主界面实际 Electron 截图覆盖 idle、running、paused、任务、统计、设置、深浅主题和任务选择器。
- 小窗实际 Electron smoke 覆盖两态、四边吸附、多显示器/DPI、320ms 收束与过渡中拖动取消。
