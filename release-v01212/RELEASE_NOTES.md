## 本次更新

### 发布校验表格式修复

- 完整继承 v0.12.11 已通过 GitHub 源码、Electron 回归、便携版主窗/小窗与安装版验收的四套艺术字体、全局强调色、日报 Dashboard、五套计时仪表、原生全屏沉浸、秒级时间之带与 `280×84` 小窗。
- v0.12.11 的两份 SHA256 数值正确，但校验表使用双空格分隔，未满足发布工作流要求的标准 `hash *文件名` 形式；旧 tag 保持不变，本版从新的干净源码提交重新生成资产并修正格式。

### 设计更新

- Noto Sans SC、霞鹜文楷、霞鹜新致宋与霞鹜漫黑四套字体全部本地嵌入；五种强调色贯穿导航、按钮、任务、统计、设置与专注，暂停保持红色。
- 统计按结论、四项指标、24 小时时间轴、任务投入排行和最近会话表顺读；多日范围显示每日趋势。
- 翻页机械修复旧数字卡住，沉浸模式使用原生全屏和 520ms 入场，时间之带支持近景/远景、暂停秒级更新与状态粒子。
- 展开小窗为 `280×84`，仍完整显示任务、当前时间、三项累计和全部控制。

## 升级提示

- 旧字体值会安全迁移到兼容字体；数据库、计时记录、任务关联与同步队列无迁移。
- 安装版可覆盖升级；便携版可直接替换可执行文件。

## 已知限制

- 番茄 To-do 当前客户端仍没有 PCRecord 远端删除与独立云端回读接口；FocusLink 只把桥明确返回 success 的记录标记为“上传已确认”。
