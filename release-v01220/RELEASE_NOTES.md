## 本次更新

- 暂停时间之带删除整块填色、红色斜纹、孔洞侵蚀和扩散椭圆，改为透明损耗区与极淡余烬轨迹。
- 每一秒先在“现在”前沿生成完整红色时间切片，再按规则网格采样成粒子，由右向左逐层剥离并向未来侧漂移、缩小和熄灭；相邻秒批次交叠，不会整秒闪切。
- 采样点脱离后分化为碎片、粉尘和短火花，带确定性湍流、重力与亮色模式对比；主时间之带和固定两态小窗复用同一模型。
- 视觉验收改用隔离 Electron 的 Canvas 与窗口 PNG 直接捕获，不操作用户鼠标键盘。
- Windows、Web/PWA 与 Android 统一升级到 0.12.20；Android `versionCode` 为 1220。

## 升级提示

- SQLite 账本、用户设置和跨设备连接配置无需迁移。
- GitHub 推送、tag 与 GitHub Release 按用户要求暂缓，本次优先提供本地安装版与便携版。

## 已知限制

- `FocusLink/cloud/` 仍是 loopback-first 测试后端，不应公开部署。
- 滴答清单与番茄 To-do 的真实投递仍只由桌面端执行。

## 验证

- format、TypeScript、ESLint、46 个测试文件共 353 项测试已通过；其余发布门禁与资产验证将在构建后补全。
- 暂停 Canvas 直接截图确认时间切片由完整采样层逐步瓦解为向未来侧漂移的粒子场，且不存在斜纹色块。
