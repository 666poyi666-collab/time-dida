## 本次更新

### 完整动态材质工作面

- 承接 v0.11.0 已完成但未在线发布的全部界面更新：低对比环境光场、轨道轮廓、稀疏粒子、状态呼吸光和强调色伴生色。
- 专注、任务、统计与设置统一使用高不透明材质面、折射边缘和定向阴影；持续动画在“减少动态效果”下停用。
- 默认字体为 Manrope Variable + Noto Sans SC Variable 的“澄澈”方案，保留 Geist“精准”方案并支持即时切换。

### 任务工作台与双同步

- 任务页直接使用滴答清单，支持搜索、强制刷新、完成/恢复、6 秒撤销，以及按最近完成、名称和截止日期排序的已完成分组。
- 滴答同步继续评论优先、稳定 marker 幂等；番茄 To-do 手动同步支持标准安装路径按需桥接，后台不擅自启动外部应用。
- 番茄目标必须通过标题与特征 API 指纹；上传 success 只表达“上传已确认”，不冒充独立云端回读。

### 过渡式边缘小窗

- 小窗严格只有折叠 `184×35` 与展开 `256×92` 两种固定尺寸；折叠时间 25px、展开时间 31px，底部为 3px 真实专注进度轨。
- 展开态集中呈现任务、当前时间、累计专注/暂停/总历时和全部当前控制。
- 原生 Windows move loop 在释放后执行“吸附 → 可见过渡 → 自动折叠”，过渡中重新拖动会取消折叠。

## 修复

- GitHub Windows runner 会把 frameless 窗口的 `window.outerHeight` 混入不可见系统边框；小窗 smoke 改由 viewport、shell 与 PNG 像素验证真实内容尺寸。
- 临时 user-data 清理加入 Windows 有界重试，避免日志尾写入的 `ENOTEMPTY` 覆盖首个错误。
- 主窗与小窗 smoke 从 `package.json` 自动推导 `release-v*`，不再硬编码旧版本目录。
- v0.11.0 公开 tag 保持不可变且不补造 Release；本补丁版本重新打包、复验并完成线上发布。

## 升级提示

- 用户数据库、任务关联、计时记录和同步队列无需迁移。
- 旧设置缺少 `fontProfile` 时自动使用“澄澈”方案，可在设置 > 体验切回 Geist“精准”。
- 开发与 Web Coding 工作目录为仓库内 `FocusLink/`；正式资产位于根级 `release-v0111/`。

## 已知限制

- 番茄 To-do 当前只能确认上传 API success，客户端没有 PCRecord 的独立云端回读与远端删除 API。

## 验证

- 最终正式包需从干净源码提交完成静态检查、28 个测试文件 / 231 项测试、生产依赖审计、构建、Electron 回归和打包。
- 需复验便携版主窗/小窗、隔离安装版主窗、双尺寸/四边过渡、六强调色、双字体和真实滴答任务链路。
