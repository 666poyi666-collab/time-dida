## 本次更新

### Linear Workbench 单一视觉系统

- 删除 Quiet / Dawn / Bloom 多主题运行时分支与自由强调色入口，保留明亮、深色、跟随系统三种外观。
- 界面蓝、专注绿、暂停琥珀固定承担各自语义；主计时使用中性墨色，状态色只出现在真正表达状态的位置。
- 主窗、任务、统计、设置与小窗共用连续、直角、低装饰的工作面，不再堆叠玻璃卡片和大圆角容器。

### 流动时间织带与专注账本

- “此刻”指针保持固定，世界时钟与刻度持续向左流动。
- 暂停时已完成专注段冻结，暂停段向“此刻”生长；继续后新专注段从指针处接力。
- 短片段保留最小可见宽度，刻度标签会避让当前指针；专注账本去除重复状态线并与主区共享同一工作面。

### 任务、统计、设置与小窗

- 任务页采用连续项目索引与任务列表，减少独立卡片和无意义装饰。
- 统计页提供真实的当天 24 小时节律、时间构成、任务去向、单次强度与混合时间轴。
- 设置页移除臃肿主题项，字体方案收敛为紧凑四列选择。
- 小窗继续严格使用折叠/展开两档共享尺寸，恢复直接的边缘进度条和高密度控制台。

## 修复

- 新增只读 `sessions.analytics(range)` 与范围交叠查询，跨午夜会话按自然日裁切，统计不再受历史列表行数限制。
- 修复 frameless 主窗口最小化、最大化/还原和关闭控制。
- 修复视觉巡检脚本可能误选 DevTools 页面的问题。

## 升级提示

- 可直接覆盖安装；本版本不迁移或删除既有专注、任务关联和同步队列数据。
- 旧主题家族与强调色设置会被兼容读取，但界面统一映射到 Linear Workbench。

## 已知限制

- 番茄 To-do 当前客户端没有 PCRecord 远端删除接口，清理能力仍仅限本地记录。
