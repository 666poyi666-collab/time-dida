## 本次更新

### 翻页、点阵与时间之带重制

- 翻页机械从零改为事件驱动的 `steady → fold → unfold → commit` 状态机。快速变化只保留最新数字，不再靠 timeout 猜测动画结束；空闲、结束和减少动态效果均直接静态提交。
- 像素点阵由 5×7 升级为高对比 7×9 整数网格并使用 crisp edges；标准仪表增加固定数字槽和 `ELAPSED / LIVE / HOLD / READY` 工业读数标记。
- 修复结束/空闲时把毫秒当秒造成时间之带高速漂移的问题。非活动态固定在最后记录锚点，变焦完成后停止持续重绘；暂停态以从当前边界侵蚀、脱落和消失的红色碎片表达节律损耗。

### 六套字体、全局强调色与紧凑小窗

- 新增霞鹜新晰黑正线体与得意黑，连同思源黑体、霞鹜文楷、霞鹜新致宋、霞鹜漫黑形成六套本地字体；它们使用不同字形骨架，不以同一字体粗细冒充选项。
- 用户强调色现在贯穿导航、按钮、字体/仪表选择态、任务、统计、专注读数、账本和时间之带；暂停继续固定为红色。
- 展开小窗收紧为 `256×70` 外框，上栏保留状态、完整任务与窗口操作，下方用主时间/60 格秒轨和三项累计/全套控制组成紧密双区。暂停粒子从真实秒轨前沿消蚀；长任务名会在字体切换后重新测量。收起态保持 `184×35` 简洁时间条。
- 应用、主界面品牌标与托盘统一为 `F / L` 双织带字标；托盘根据系统明暗自动选择前景，运行使用专注色、暂停使用暂停红。

### 可读的统计 Dashboard

- 统计页按“结论 → 四项 KPI → 时间节律 → 任务去向 → 暂停损耗”顺读，不在 Dashboard 内重复最近会话账本。
- 单日视图同时提供固定 00/06/12/18/24 全天定位轨和自动放大的专注/暂停双车道；旧记录缺少片段明细时显示明确说明。
- 多日视图使用带时长刻度的专注/暂停堆叠日柱，每天可悬停或键盘聚焦读取精确值；任务去向改为百分比合计恰好 100% 的构成带，并分开已关联、未关联和旧会话汇总。
- 删除约 750 行已经没有入口的 weave、matrix、beads、mosaic 与旧 Analytics Dashboard 样式及纯函数。

### 番茄 To-do 一致性

- 已有 marker 的单个或批量学科修改若因桥不可用或上传失败未写入外部记录，会进入 durable queue；队列清空前不会把旧学科记录的 `isSynced=1` 误显示为新学科“上传已确认”。
- 删除确认现在明确说明：番茄 To-do 只清理本机记录，当前客户端不支持远端删除。

## 修复

- 修复翻页动画在快速连续变化和 reduced-motion 下可能停在半片数字的问题。
- 修复时间之带在结束/空闲状态继续漂移与持续 60fps 空转的问题。
- 修复小窗暂停粒子固定从轨道最右端喷出、结束态只显示最后一个专注片段、底栏按钮换行以及字体切换后任务名未重新测量的问题。
- 修复旧 Dashboard 任务排行被同名 `.mosaic` 规则裁切、任务构成整数百分比可能累计超过 100%、未关联与旧记录被误画成强调色的问题。
- 修复番茄 To-do 学科外部更新失败后没有持久补传意图的问题。

## 升级提示

- 无数据库迁移；专注记录、任务关联与两个同步队列保持原样。
- 新增两套本地字体资源，旧字体偏好继续保留；未知旧值安全回落到 Noto Sans SC。
- 旧小窗尺寸会自动归一到收起 `184×35` 或展开 `256×70`，不增加第三种尺寸。
- 安装版可覆盖升级；便携版继续使用可执行文件旁的 `focuslink-data/`。

## 已知限制

- “上传已确认”表示番茄 To-do 的 `cloudSyncUploadRecord` 明确返回 success 并成功持久化本地同步状态；当前 1.6.2 客户端没有 PCRecord 独立云端回读 API。
- 番茄 To-do 当前没有 PCRecord 远端删除 API，删除只验证 `local-record-only`，`remoteDeleteSupported=false`、`remoteCleanupVerified=false`。
