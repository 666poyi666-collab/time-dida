## 本次更新

### 「时间仪器 Time Instrument」设计系统重建

- 整窗收敛为单一浅色工作面，区域之间只有 1px 发丝线，删除圆角卡片墙、玻璃拟态与大面积渐变。
- 移除整套旧 temporal-ui 主题层、AmbientField 环境动效与 fontProfile 字体气质机制；主题只保留明亮 / 深色 / 跟随系统。
- 颜色收敛为四种语义：界面蓝 = 操作、专注绿 = 运行（翡翠/森林/薄荷/青碧四档可选）、暂停红 = 损耗、深红 = 危险操作；主计时暂停时以红色显示本次暂停经过。
- 字体系统收敛为本地嵌入的 Geist Variable（界面拉丁）+ MiSans（中文）+ JetBrains Mono（数据与账本），仪表字形使用 Inter Tight 与 Oswald，像素数字为自绘点阵。

### 四套计时仪表

- 标准等宽 standard、翻页机械 flip（上下分片 + 中央转轴翻页）、像素点阵 pixel（点阵数字 + 随累计专注点亮的专注核心）、极细编辑 thin，四种样式在设置页以真实渲染实时预览，选择即保存。
- 旧计时样式设置（editorial / digital / mono）启动时自动迁移为 thin / pixel / standard，未知值回落标准样式，不会出现白屏。

### 时间之带重写为 canvas

- 刻度即进度：颜色填充整个刻度区域，分钟/秒刻度绘于颜色之上，不再有独立的第二进度条。
- 专注时进入秒级近景，逐秒 130ms 离散步进；暂停时 720ms 对数变焦拉远到约 30 分钟大格远景；继续时第一帧状态与主读数即变绿，变焦只负责镜头拉近。
- 支持 reduced-motion：瞬时切换尺度，每秒数据更新不丢失。

### 统计 Dashboard 重做

- 顶部先给一句自然语言结论，进行中会话计入并明确标注「进行中」。
- 单日为 24 小时时间织带（专注/暂停/空白分明，悬停查看精确片段）；多日为日期×时段节律矩阵；单次会话质量为珠链图（开始时刻 × 时长，颜色表达专注率与打断，点击展开）；时间去向为马赛克比例带 + 精确时长占比行，「未关联」是明确的最后一类。
- 范围在单日 / 近 7 天 / 半个月 / 1 个月 / 自定义之间切换，旧请求的响应不会覆盖新结果；不再有环形图与 KPI 卡片墙。

### 沉浸模式与专注页

- 沉浸覆盖层重排：当前任务、状态、四套仪表之一、累计专注/暂停/总历时、全部控制，以及放大约 1/4 屏高的时间之带；按 Esc 或右上角按钮退出，计时不中断。
- 专注页删除左侧蓝色选中竖线；右侧专注账本与主区共享同一工作面，仅以 1px 发丝线分隔。
- 按钮系统按语义分离：开始/继续为操作蓝主按钮，暂停为深色，结束为次级描边；大窗口重新排版，不再留白。

### 小窗修复

- 仍严格只有收起 `184×35` 与展开 `320×116` 两个固定尺寸，唯一来源为 `shared/miniWindowLayout.ts`；旧 `320×124` 设置自动归一。
- 展开态六区重排：完整任务名（无省略号、无渐隐，极长名克制滚动）、当前时间、累计专注、累计暂停、总历时与全部控制；当前时间与继续按钮分属独立行，不再互相重叠。
- 边缘吸附、320ms 收束后自动折叠、多显示器与 DPI 行为保持既有契约。

## 修复

- 修复小窗展开态主时间与「继续」按钮拥挤重叠、任务名被省略号或渐隐截断的问题。
- 修复旧主题/字体/计时样式设置在新版本启动时可能失效的问题：旧字段仅作兼容解析，计时样式自动映射到四套新仪表。
- 修复统计范围快速切换时旧响应残留覆盖的可能路径，请求版本保护覆盖全部范围预设。

## 升级提示

- 可直接覆盖安装；本版本不迁移或删除既有专注记录、任务关联与同步队列数据。
- 旧计时样式（editorial / digital / mono）自动迁移为 thin / pixel / standard；旧主题与字体偏好字段仅保留兼容解析，运行时不再消费。
- 小窗旧尺寸设置（含 `320×124`）自动归一到最近的合法预设。
- 安装版与便携版升级注意事项与以往一致：便携版数据保留在可执行文件同目录的 `focuslink-data/`。
