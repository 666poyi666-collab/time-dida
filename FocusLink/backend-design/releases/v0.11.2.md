## 本次更新

### 连续专注工作台

- 计时器与本次片段账本合并为一个连续材质工作台，移除两个独立大框和活动片段内嵌卡片造成的割裂。
- 计时核心加入多层轨道、轴线、状态节点和节奏信号；运行、暂停与 reduced-motion 使用不同但克制的运动状态。
- 页面切换重建为与导航顺序一致的短距离空间过渡，避免突兀闪切和全屏滤镜合成问题。

### 真实统计图谱与字体分型

- 统计页按真实筛选结果生成专注/暂停环形构成、每日专注柱状趋势和最长五次横向排行；空数据不展示伪造图形。
- 字体方案改为“舒展”与“锐界”：前者使用 Manrope + Noto Sans SC，后者使用 Geist + Microsoft YaHei UI，并分别调整正文及计时数字的字重与字距。
- 桌面 UI smoke 新增三类统计图存在性及 computed font family / tracking 差异断言，防止视觉能力再次退化。

## 修复

- Windows UI smoke 清理临时日志目录时使用有界递归重试，避免 Electron 尾写日志产生 `ENOTEMPTY` 并掩盖已通过的界面断言。
- 保持任务完成/恢复、滴答 CLI、番茄 To-do、小窗固定两态与三时间账本语义不变。

## 升级提示

- 用户数据库、任务关联、计时记录、同步队列与小窗尺寸无需迁移。
- 原字体设置会自动映射到对应的新视觉名称，不丢失偏好。

## 已知限制

- 番茄 To-do 仍只能确认上传接口 success，不能声明客户端未提供的独立云端回读或远端删除。
- 无已知阻断问题。
