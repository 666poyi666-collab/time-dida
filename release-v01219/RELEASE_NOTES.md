>

## 本次更新

- 设置页改成连续、规整的工作区；界面字体、全局强调色、计时仪表与状态语义分组清晰，仪表预览使用固定舞台，标准等宽读数不再被裁切。
- 暂停效果重制为真正的粒子消散：碎片、尘点和短火花连续发射，相邻秒批次重叠，并配合余烬光与双层扩散波；15 秒观察舱展示秒级细节，孔洞只保留为弱材料纹理。
- 小窗暂停粒子改由真实暂停时长连续驱动，稳定粒子身份跨帧移动；不再依靠无限 CSS 循环，也不会在整秒边界整批跳变。
- 专注账本的每条专注记录都使用全局强调色，每条暂停记录都使用暂停红；当前条目只增加背景强调。
- 前端与后端档案入口重新分类，产品交互、维护、架构、测试发布和历史 Release 各归其位。
- Windows、Web/PWA 与 Android 统一升级到 0.12.19；Android `versionCode` 为 1219。

## 升级提示

- SQLite 账本、用户设置和跨设备连接配置无需迁移。
- GitHub 推送、tag 与 GitHub Release 按用户要求暂缓，本次优先提供本地安装版与便携版。

## 已知限制

- `FocusLink/cloud/` 仍是 loopback-first 测试后端，不应公开部署。
- 滴答清单与番茄 To-do 的真实投递仍只由桌面端执行。
