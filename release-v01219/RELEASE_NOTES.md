# FocusLink v0.12.19

> 发布日期：2026-07-20
>
> 对应源码：`bc66283`
>
> 发布类型：正式版
>
> 验证状态：已通过

## 本次更新

- 设置页改成连续、规整的工作区；界面字体、全局强调色、计时仪表与状态语义分组清晰，仪表预览使用固定舞台，标准等宽读数不再被裁切。
- 暂停效果重制为真正的粒子消散：碎片、尘点和短火花直接从时间材料与“现在”前沿连续脱离，相邻秒批次重叠，并配合余烬光与双层扩散波；不再叠加独立观察框，孔洞只保留为弱材料纹理。
- 小窗暂停粒子改由真实暂停时长连续驱动，稳定粒子身份跨帧移动；不再依靠无限 CSS 循环，也不会在整秒边界整批跳变。
- 专注账本的每条专注记录都使用全局强调色，每条暂停记录都使用暂停红；当前条目只增加背景强调。
- 前端与后端档案入口重新分类，产品交互、维护、架构、测试发布和历史 Release 各归其位。
- Windows、Web/PWA 与 Android 统一升级到 0.12.19；Android `versionCode` 为 1219。

## 升级提示

- SQLite 账本、用户设置和跨设备连接配置无需迁移。
- 本版可直接从 v0.12.18 升级；安装版与便携版使用同一源码提交构建。

## 已知限制

- `FocusLink/cloud/` 仍是 loopback-first 测试后端，不应公开部署。
- 滴答清单与番茄 To-do 的真实投递仍只由桌面端执行。

## 验证

- format、TypeScript、ESLint、46 个测试文件共 354 项测试、audit 0 漏洞、build、dist、Electron 崩溃恢复与 Android unit/lint 全部通过。
- 解包版与便携版通过完整主窗 smoke；小窗通过 `184×35 / 256×70`、四边吸附、主题切换与暂停粒子连续帧验证。安装版与便携版内嵌版本均核验为 `0.12.19 / bc66283`，并成功创建 Renderer target；原 0.12.18 的进程、注册项与快捷方式全程保留。
